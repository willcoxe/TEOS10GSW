# TEOS10GSW

[![Build Status](https://travis-ci.org/willcoxe/TEOS10GSW.svg?branch=master)](https://travis-ci.org/willcoxe/TEOS10GSW)

This repository builds binary artifacts for the GSW project. Binary artifacts are automatically uploaded to
[this repository's GitHub releases page](https://github.com/willcoxe/TEOS10GSW/releases) whenever a tag is created
on this repository.

This repository was created using [BinaryBuilder.jl](https://github.com/JuliaPackaging/BinaryBuilder.jl)
